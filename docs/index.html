Quick Notes

  • All nodes can be connected. Meaning no node has any particular data type.
  • There is one small note to above. Output Nodes when connected to the Erosion Node, will give more options for which layers to convert to images. This is done for convinience, so that the user does not have to connect multiple outputs. More on this here.
  • Make sure to select 'Fake User' (looks like a Shield Icon) to save the node tree, otherwise it might get lost. More info here
