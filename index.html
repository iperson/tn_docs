GPU Accelerated Add-on for Blender 3D

Fast GPU Erosion

Terrain Nodes uses the power of your NVIDIA GPU to quickly perform complex erosion computations (compute capibility 3.5 and above is required).

Flexible Node UI

Node interface allows you to setup and reuse complex terrain generation processes.

Live Preview

Switch between different layers to see what is happening during erosion.

Sculpt Erode Repeat

Erode sculpted terrains and sculpt again.
